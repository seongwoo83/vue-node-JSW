​‌‌‍⁡⁢⁣⁣[ 01. 뷰엑스 설치하기 ]⁡​

1. 기본 webpack에는 Vuex라이브러리가 설치되어 있지 않음
2. Vue CLI로 설치명령어를 실행하여 설치 필요
    ((기본 npm 설치 명령어)) - npm install 설치할패키지이름@버전
    npm install --save --force vuex@3 babel-polyfill
    1) 기본 명령어 : npm install
    2) 옵션1(필수) --save : 설치레코드 저장
        -> package.json의 "dependencies"에 기록해서 관리할 수 있도록 함
            (현재 최신 모듈은 --save를 쓰지 않아도 자동으로 등록해줌)
    3) 옵션2 --force : 강제 설치
        -> 기존같은 종류가 설치된 것을 덮어쓸때, 강제설치 필요시
    4) vuex@3 : vuex 설치시 3.x 버전으로 설함함
        (만약 버전을 안쓰면 최신버전을 자동으로 설치함)
    5) babel-polyfill : babel은 브라우저 호환성 때문에 설치함
        -> 오래된 브라우저도 최신 코드가 실행되도록 변환해줌 - 뷰엑스와 함께 설치됨
        -> 구체적으로 ES5, ES6, JS문법을 이해못하는 브라우저에 적용되도록 문법적 변환해주기때문에 예전 JS문법을 작성할 필요가 사라짐
